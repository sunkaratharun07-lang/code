this program helps a shopkeeper keep track of products.

it uses parallel arrays to save information for each item.

only 10 items can be stored because the arrays are fixed.

each item includes an id, a name, a quantity, and a price.

a menu appears again and again to guide the user.

the user can enter new product details when needed.

all existing products can be shown in a simple list.

the program adds up the worth of the entire inventory.

it also checks which item is worth the most and which is worth the least.

the search option allows the user to find an item by its id.
