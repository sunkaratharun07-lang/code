the program asks how many students are there in the class, maximum allowed is 10


it uses a simple one dimensional array to store the marks of all students


it takes marks input one by one for each student using a loop


it adds up all marks to find the total marks of the class


it keeps checking which mark is the highest and which is the lowest while reading input


it counts how many students passed and how many failed based on their marks


it decides grades like a+, a, b, c, d or fail depending on the mark range


it counts how many students belong to each grade group


it calculates the average marks of the class by dividing total marks by number of students


it displays the summary showing average, highest, lowest, number passed, failed and grade counts
