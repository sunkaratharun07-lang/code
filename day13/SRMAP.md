he program is designed to calculate smart bus fares for the srm university shuttle system.

it should take input for multiple passengers one by one using loops.

the fare depends on distance, passenger age, and time of travel.

the base fare is ₹10 per kilometer for all passengers.

children below 12 years get a 50% discount, and senior citizens aged 60 or above get a 30% discount.

a 20% peak hour surcharge is added between 8–10 am and 5–8 pm.

the minimum fare should always be ₹20 even after applying discounts.

the program should display each passenger’s fare details after calculation.

at the end, it must show a daily summary with total passengers, total revenue, highest fare, and lowest fare.

the code must use only loops, conditional statements, and primitive data types without arrays or functions.
