the program starts by asking the user to enter the student’s name, registration number, and section.

it then asks how many employees there are (not more than 10).

the user enters the salary for each employee, which are stored in an array.

a menu is shown with options to perform different salary analyses.

option 1 displays all the entered salaries.

option 2 calculates and shows the total and average salary.

option 3 finds and displays the highest and lowest salary.

option 4 counts how many employees earn above or below the average salary.

option 5 sorts the salaries in ascending order using the bubble sort method.

the program repeats the menu until the user chooses exit and displays the student’s details after every result.
